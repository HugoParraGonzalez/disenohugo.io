Como introducir fuentes personalizadas

Para introducir fuentes personalizadas se ha de a√±adir varias etiquetas de tipo link en el head del documento html

<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Cormorant:wght@300&family=Rubik:wght@300&display=swap"
rel="stylesheet">

Despues en el estilo de la pagina declaramos la fuente que deseemos para cada caso

body {

font-family: 'Cormorant', serif;
}

h1 {

font-family: 'Rubik', sans-serif;
}
